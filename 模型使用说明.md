## 🚀 使用步骤
1. 安装依赖包: `pip install tensorflow pandas scikit-learn numpy matplotlib`
2. 加载模型和相关文件
3. 使用 `predict_model.py` 进行预测
4. 参考 `model_usage_example.py` 了解详细用法

## ⚠️ 注意事项
- 输入文本需要经过相同的清洗处理
- 选项类型需要匹配训练数据中的类型
- 模型预测结果仅供参考，置信度<0.5时建议人工确认
